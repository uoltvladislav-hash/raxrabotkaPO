  Решатель квадратных уравнений

Простое и удобное приложение для решения квадратных уравнений вида **ax² + bx + c = 0** с графическим интерфейсом.

✨ Возможности

- 🔢 Ввод коэффициентов a, b, c
- 📐 Автоматическое вычисление дискриминанта
- 🎯 Нахождение корней уравнения
- 🖥 Понятный графический интерфейс
- ⚡ Мгновенный результат

## 🚀 Установка и запуск

### Способ 1: Готовый EXE файл (для Windows)
1. Скачайте `grafinterface.exe` 
2. Запустите файл (Windows может показать предупреждение - нажмите "Подробнее" → "Выполнить в любом случае")
3. Готово! Программа работает без установки Python

### Способ 2: Запуск из исходного кода
